## Project 2 - Manhattan Sales Volume


This project is comprised of 5 Jupyter Notebooks


- Sales Data Cleaning
- NYC Crime Data Cleaning
- Permit Data Cleaning
- Scraper
- Master DataFrame

3 Notebooks are for cleaning/formatting each data set and exporting to a csv to be merged in the Master DataFrame. The scraper notebook is an embed of the Sales Data Cleaning notebook, and passes to it relevant location data scraped from google maps. 

The Master dataframe is where each data set is merged and formatted on location, and a simple model is built.
